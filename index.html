    CSS Box Model

    The CSS Box Model is the main layout of almost all webpages. Using a CSS box model will allow the user to space out different objects from one another. Additionally, padding can be added to an object, creating the object to stand out more, or a border can be used instead for the same purpose. Lastly, a margin can be added to the website to prevent text, photos, and other content, from flowing off of the main page.


    Content - This is held within the centre of the box model and contains the main information, like text and images, that are to be seen by the public


    Padding - Padding is used as an invisible barrier to prevent objects like text and images from interferring and contacting with each other


    Border - The Border is placed around the padding to add a bit more creativity to a website, if the developer chooses to do so


    Margin - Lastly, the margin is a clear space around everything which serves the purpose of ensureing that all text and other content can be viewed without it slipping off to the side
