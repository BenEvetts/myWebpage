    How HTML accesses CSS

    HTML is always the base coding to any website. Also, it can be used to implement a variety of other types of code, one of these are CSS. For the CSS to be incorporated you must use one (or more) of the listed method

    1. Internal
    2. Inline
    3. External

    Internal - This is when the code is written within the "head" of a web page. If used, the entire page can be changed, although this could be a disadvantage for the reason that it may take up a larger file size compared to the other methods.


    Inline - To many this is considered as the easiest and simplest form of adding a style, because the CSS is applied directly to a HTML tag. To change a CSS style, one would require to access the HTML and search the file for the style attribute however, causing difficulty in this situation.


    External - External is my preferred method for the reason that it can contain all code neatly in a separate file (most commonly called "mystyle.css" or "stylesheet.css") with a ".css" on the end. From that point on the user will have the ability to identify different segments, or individual pieces, and then edit it's color, position, and more!

    CSS Box Model

    The CSS Box Model is the main layout of almost all webpages. The purpose of it is to allow the user to choose locations for certain objects with a bit more ease. You can add padding to make things stand out and add a border between different things. And you can add a margin to stop text and photos from going to the edge of the website.


    The content box is in the centre of the page between the margin, border and padding. The content is the stuff like text, photos and graphs.


    Padding is placed around the content to create and invisible barrier to stop text and photos colliding.


    A border is placed around the padding which is around the content. The border adds a solid color so that it looks different. The border can just be a base color the same as the background or be a different color to make it look nicer.


    The margin is a bit of code that clears a space around the area of the website. The margin is a clear area so makes no difference to the background.
